# Simple Voting App

A Web3.0 voting app on blockchain with Next.js and Ethers.js, using Sepolia testnet.

## Setup
1. `npm install`
2. Update `src/lib/eth.ts` with contract address and ABI.
3. `npm run dev` for local.

## Deployment
- Deploy contract via Remix to Sepolia.
- GitHub push.
- Vercel deploy.
